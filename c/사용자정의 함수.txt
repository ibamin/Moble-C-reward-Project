반환형에 들어갈 수 있는 것 : 반환형 / void(empty) 반환될 값의 자료형
매개변수에 들어갈 수 있는 것 : 변수의 자료형(int x와 같은 형태) / void(empty)

3. 특수기호가 들어있는 이름

2 반환형 : 0 매개변수 : X (변수의 초기화값을 이력받아 반환하는 함수를 만들때 사용됨)
int get_int(){
  int x;
printf("정수 한개 입력 : ");
scanf("%d",&x);
return x;
}

int main(){
int x,y;
x=get_int(); //x를 get_int라는 함수를 통해 초기화
y=get_int();
printf("%d",x+y);
}

3 반환형 :X 매개변수 : 0
두개의 정수를 전달 받아 합을 구하고 출력하는 함수 제작 예시
void sum (int x,int y){
printf("%d + %d = %d",x,y,x+y);
}

int main(){
int x,y;
printf("두개 정수 입력 : " );
scanf("%d %d",&x,&y);
sum(x,y);
}
